## 문제 1
### final, const
> 코드를 실행하는 오류가 발생하였다. 오류가 발생한 구간 윗줄에 오류원인을 주석으로 작성하시오

```dart
void main() {
  final DateTime now = DateTime.now();
  print(now);
  
  const DateTime now2 = DateTime.now();
  print(now2);
}
```
