## 문제 2
반복문을 사용하여 별을 출력하시오

```dart
// 출력 내용(문자열 앞에 공백은 없다.)
// 1
// 2
// 3
// 4
// 5
// 4
// 3
// 2
// 1

const int N = 5;

void main() {
  for (int i = 1; i < N + 1; i++) {
    print(i);
  }
  for (int i = N - 1; i > 0; i--) {
    print(i);
  }
}
```
